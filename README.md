# 🎓 NextLevel
Plataforma educativa desarrollada en **ASP.NET MVC** para la gestión de cursos, docentes y estudiantes.

## 🧩 Funcionalidades implementadas hasta el momento
- Registro y autenticación de usuarios  
- Previsualización y búsqueda de cursos  
- Sistema de mensajería interna  
- Clases en vivo y gestión de materiales
